🔐 What is Secureption?

Secureption is a beginner-friendly, web-based RSA encryption and decryption tool built using Python and Flask. It helps users understand how public-key cryptography works by allowing them to generate keys, encrypt, and decrypt messages directly in the browser.


📌 Key Features


  • 🔑 RSA Key Generation: Create 2048-bit secure key pairs
  • 📥 Encryption: Encrypt plaintext using the public key
  • 📤 Decryption: Decrypt messages using the private key
  • 📄 Base64 Output: Easy to share and readable ciphertext
  • 🧠 Educational Focus: Built for students and security learners
  • ⚙️ Secure Library: Uses Python's cryptography with OAEP padding


🔄 RSA Key Exchange Overview

A high-level visualization of how RSA encrypts and decrypts messages securely.


This diagram demonstrates the flow of information in RSA: the sender encrypts a message using the recipient's public key. Only the matching private key can decrypt it, ensuring confidentiality. Even if intercepted, the ciphertext cannot be decoded without the private key.
