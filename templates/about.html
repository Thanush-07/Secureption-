About Us

Secureption is a simple web-based RSA encryption tool built with Python and Flask to help beginners understand public-key cryptography. Users input a message and a PEM-formatted public key, and the app securely encrypts the message using RSA with OAEP padding. The result is displayed in Base64 format for easy sharing. This project demonstrates the encryption flow clearly, using the `cryptography` library for secure operations and Flask for the web interface.

The process begins with generating an RSA key pair (a public and private key). The sender encrypts a message using the recipientâ€™s public key. Only the private key can decrypt the message. This ensures that even if the data is intercepted, it remains unreadable to anyone without the private key.
